# أنـتـي - منصة تواصل اجتماعي

**أنـتـي** هي منصة تواصل اجتماعي عربية، مصممة لربط المستخدمين في بيئة آمنة ومريحة.

## الميزات (لاحقًا):
- حسابات شخصية
- نشر منشورات وصور
- رسائل خاصة
- دعم المواضيع الحساسة والخصوصية
- تصميم يدعم اللغة العربية بالكامل
